The basic language of the web: HTML & CSS

Posted by Luara Joneson, June 4th 2023

All modern websites are built using three fundamental technologies: HTML, CSS and JavaScript.
These are the languages of the web. In this post, let's focus on HTML.
We will learn what HTML is all about, and why you too should learn it.

What is HTML

HTML stands for HyperText Markup Language. It's a markup language that web developers use to structure and describe
the content of a webpage (not a programming language). HTML consists of elements that describe different types of content:
paragraphs, links, headings, images, video, etc. Web browsers understand HTML and render HTML code as websites.

In HTML, each element is made up of 3 parts:

  1. Opening tag
  2. Closing tag
  3. Elements
